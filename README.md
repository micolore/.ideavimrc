# .ideavimrc

* 目标
 * 让你写代码的时候尽量不用鼠标，一个键盘搞定。
* 配置原则
  * 常用的优先配置，不常用的可以不进行配置。循序渐进，不用一下配置很多不常用（配了也记不住）的配置。
